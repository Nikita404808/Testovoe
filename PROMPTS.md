# PROMPTS

Ниже примеры рабочих промптов, использованных при реализации проекта.

1. Сформировать архитектуру минимального веб-приложения для ролей `dispatcher` и `master` с SQLite и локальным запуском.
2. Описать таблицы БД и миграции для `users`, `requests`, `request_events` с нужными статусами.
3. Реализовать безопасный переход `assigned -> in_progress`, чтобы при параллельных запросах один получал успех, второй `409`.
4. Подготовить UI-страницы: создание заявки, панель диспетчера, панель мастера.
5. Добавить автотесты на создание заявки и race-сценарий для `take`.
6. Подготовить README с запуском через Docker Compose и инструкцией проверки гонки через `curl`/скрипт.

Все решения и код были проверены и адаптированы под конкретный проект вручную.
